How to create a local server (nodejs) :


-Step1 : Install NodeJs.
https://nodejs.org/en/download/

-Step2 : Click on "Run_Local_Server.bat".


Now you can acces server files (only files inside www folder) by typing
http://127.0.0.1:9000/filename


Ex : http://127.0.0.1:9000/mr-mushroom/game-data.json
